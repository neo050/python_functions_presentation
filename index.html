ברוכים הבאים!

היום נלמד איך לכתוב קוד מקצועי יותר

🎯 מטרות השיעור

  • להבין מהי פונקציה ולמה אנחנו צריכים אותה
  • ללמוד את המבנה הבסיסי של פונקציה
  • לתרגל כתיבת פונקציה פשוטה
  • להבין את המושגים קלט ופלט

🤔 למה אנחנו צריכים פונקציות?

דמיינו שאתם רוצים לחשב ממוצע של ציונים

# Without functions - we need to write this code again and again:
sum1 = 85 + 90 + 78 + 91
average1 = sum1 / 4
sum2 = 92 + 88 + 95
average2 = sum2 / 3
# This is not efficient! 😫

פונקציות עוזרות לנו לא לחזור על עצמנו!

📦 מהי פונקציה?

פונקציה היא קטע קוד שמבצע משימה מסוימת
אנחנו נותנים לה מידע (קלט)
והיא מחזירה לנו תוצאה (פלט)

📥 ⬅️ 🔧 ⬅️ 📤

קלט⬅️ עיבוד ⬅️⬅ פלט

🔄 קלט ופלט - הסבר מפורט

📥 קלט (Input)

המידע שאנחנו "מזינים" לפונקציה

נקרא גם פרמטרים

📤 פלט (Output)

התוצאה שהפונקציה מחזירה לנו

נקרא גם ערך מוחזר

כמו מכונת קפה: אתם מכניסים כסף וקפה ☕
המכונה מחזירה לכם כוס קפה מוכנה! ✨

⚡ תחביר - איך כותבים פונקציה?

def function_name(parameter1, parameter2):
     """Description of what the function does"""
    # Function code goes here
    return result
  • def - מילה שמגדירה פונקציה חדשה
  • function_name - שם ייחודי שנבחר
  • () - סוגריים שמכילים את הפרמטרים
  • : - נקודותיים שמסמנות תחילת הפונקציה
  • return - מחזיר את התוצאה

💪 בואו נתרגל!

כתבו פונקציה שמקבלת שני מספרים
ומחזירה את המספר עם הערך הגדול יותר 
def find_max(num1, num2):
"""Function that returns the larger number"""
    if num1 > num2:
     return num1
    else:
     return num2
# How to use the function :
result = find_max(10, 5)
print(result)
# Output: 10

🎉 סיכום

  • פונקציה - קטע קוד שמבצע משימה מסוימת
  • קלט - המידע שאנחנו נותנים לפונקציה
  • פלט - התוצאה שהפונקציה מחזירה
  • def - המילה שמגדירה פונקציה חדשה
  • return - מחזיר את התוצאה
